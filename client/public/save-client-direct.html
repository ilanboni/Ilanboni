Salvataggio Diretto Cliente

Questo tool permette di inserire un cliente direttamente nel database usando una query SQL diretta.

Istruzioni

  1. Modifica i dati cliente nel JSON qui sotto come necessario
  2. Clicca "Salva Cliente" per inviare la richiesta al server
  3. Verifica il risultato nella sezione "Risposta"
